# 更新履歴

## v1.3 (2026/03/15)
- 同期状態をヘッダーで分かりやすく表示（同期済: 緑 / 同期中: 黄 / オフライン: 赤）
- 同期状態にランプ表示を追加し、ひと目で状態を確認しやすく改善
- オフライン時も端末側に変更を保持し、通信復帰後に自動で再同期するよう改善
- 同じデータを別端末で編集したとき、競合を検知して選択できるよう改善

## v1.2 (2026/03/15)
- データ保存方式を変更し、端末をまたいで同じデータを使いやすく
- 初回利用時に専用ID（UUID）を自動発行するように
- 専用ID（UUID）を入力すると、対応するデータを開けるように
- 一度取得した専用ID（UUID）は端末に記録されるため、通常は再入力不要
- JSONバックアップ / 復元機能を削除

- こことサイドパネルを実装
